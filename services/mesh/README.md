# Mesh (Python)

The agent runtime. Owns:

- Research agent orchestration and prompt management
- LLM routing — cheap model for triage/search, stronger model for synthesis and contradiction resolution
- Embeddings, semantic search, and source-credibility scoring
- Contradiction detection across agents' claims
- The evaluation harness scoring agent output quality
